משחק טרמינלי "להציל  את המלך"-
	מטרת המשחק היא להביא את המלך אל כס המלכות ובמהלך הדרך הוא נעזר בפמליתו 	כדי לצלוח מכשולים שמונעים ממנו להגיע אליו.                                                                  	בקובץ השלבים הנוכחי ישנם 3 שלבים שנטענים אוטומטית כאשר המלך מגיע אל כסאו
	אריחי המשחק:  גבולות לוח המשחק - ' // ' 
		           חומה - '=' הדמויות לא יכולות לעבור דרכה
		           אירח ריק - '  ' כל דמות יכולה להתהלך עליו.
			אורק - ' ! ' - רק דמות הלוחם יכולה לעבור עליו ו"להילחם בו" לאחר 					      המעבר נופל מהאורק מפתח F.
			כס המלכות - '@' - רק המלך יכול לשבת עליו ובכך מסתיים השלב
			אש - ' * ' - רק דמות המכשף יכולה "לכבות" את האש 
			שער - ' # ' - רק דמות הגנב יכולה לפתוח את השער ובכך להעלים אותו, 				      אך רק כאשר הגנב אסף את המפתח 
			טלפורט - ' X ' - אריח שמאפשר מעבר ממיקומו למיקום טלפורט אחר על 					לוח המשחק , דמויות המלך ,לוחם וגנב יכולים לעבור 					בעזרתו, המכשף לא יוכל לעבור בעזרתו למיקום אחר.
			מפתח - 'F' - רק הגנב יכול לקחתו כל דמות אחרת יכולה לעמוד עליו אך				       לא לקחתו

                      דמויות: מלך -  'K' - נעזר בשאר הדומיות כדי לצלוח מכשולים ולהגיע לכס המלכות.
		         מכשף - 'M' - יכולת מיוחדת - כיבוי אש
		         גנב -     'T' - יכולת מיוחדת - פתיחת שערים אם יש לו מפתח, מעבר 				         בטלפורט
		         לוחם -  'W' - יכולת מיוחדת - להילחם באורק ותמיד לנצח, מעבר בטלפורט

רשימת קבצים:   עבור על דמות בניתי מחלקה משלה קובץ הדר וקובץ cpp 
קובץ מיקום     (Location)  - משמש כדי לדעת מיקומים על הלוח עבור כל אריח או דמות.
 קובץ שימושים (utilities)   - מכיל משתני enum עבור לחצנים ואריחים על הלוח.
קובץ משחק     (game)     - מנהל את כל המחלקות ומריץ את לוח המשחק.
קובץ לוח         (board)     - מכיל את לוח המשחק שמנהל את כל הקשור ללוח המשחק 					ותזוזות הדמויות על ידי קובץ המשק
מבני נתונים עיקריים ותפקידיהם:
